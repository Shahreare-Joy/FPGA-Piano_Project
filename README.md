## Overview

This repository contains documents of an FPGA-based piano implemented using SystemVerilog.

Seven touch sensors are connected to the Arduino pins 0 to 6 as piano keys. When a key is touched, the FPGA generates the corresponding musical tone and outputs it to a buzzer. The note being played is also displayed on the seven-segment displays. Currently, it covers octave 4 of the piano keybord and doesn't cover the the flats or sharps (the black keys). The files are organized in the repository.
